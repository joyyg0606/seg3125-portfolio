About Me

Hi, my name is Joy Yeung. I am a Software Engineering student at the University of Ottawa. I am still learning UI and UX design, so this portfolio is also a record of my progress.

For this course, I am trying to focus on making pages that are clear first. I care about spacing, readable text, simple navigation, and making sure the user can understand what to do next.

My Assignment 2 project is a hair studio website called Bloom & Brush. I chose it because appointment booking and service browsing are both easy user goals to test in a prototype.

How I Work

Start With the User

I try to write down what the user is trying to do before choosing colours or layout. For Assignment 2, the two main goals were booking an appointment and comparing services.

Make a Rough Version

I use simple mockups first because it is easier to change the flow before spending time on the final page. This helped me separate the booking flow from the service browsing flow.

Check What Feels Confusing

After the first version, I look for places where the user might hesitate, like unclear buttons, missing prices, or too many choices on one screen.

Assignment 3 Notes

For the memory game, I focused on two realistic user needs: someone who wants a simple memory game, and someone who wants a harder challenge with more options.

The final prototype combines both paths by letting users choose a level, pick a tile theme, watch a pattern, and repeat it back by clicking the tiles.

Prototype checks

  • The memory game name and purpose are shown.
  • Users can choose a level and tile theme.
  • The memory game links from this portfolio.
  • The game board reacts when users repeat the pattern.
